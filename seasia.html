Politics at the Border


Thongchai's epistemological approach to nationhood is also a critique visual representation. This particular map was embedded into the page through OpenLayers, an open source javascript library that allows for displaying a map using latitude and longtitude coordinates. I'm intentionally using it here to reflect on Thongchai's commentary on the technologies of territoriality, even though he emphasizes the circulation of maps in textbooks, theater performances and what he calls the "database of history." You can use the following code to get started. 

 
var map = new ol.Map({
    target: 'map',
    layers: [
        new ol.layer.Tile({
            source: new ol.source.OSM()
        })
    ],
    view: new ol.View({
        center: ol.proj.fromLonLat([104.8455, 14.0086]),
        zoom: 8
    })
});
